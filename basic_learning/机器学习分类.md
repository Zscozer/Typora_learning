## 	机器学习训练步骤

### 		1.Fuction with unknown parameters

​			 Model: y = w*x+b (w,b就是未知参数)

​			其中x,y这些已知的东西叫feature

​			w: weight

​			b: bias

### 		2.Define loss from Training data

​			loss is a function of parameters L(w,b)

​			A easy define of Loss: how good a set of values is 

### 		3.Optimization 

​			w^,b^= argmin L
